## ü§ñ Robot Mascot: "Craft Bot"

### Character Overview
**Name**: Craft Bot (or "Crafty")
**Personality**: Encouraging, quirky, calm
**Role**: Guides users through features, celebrates wins, provides friendly assistance

### Visual Specifications

#### Base Design
- **Body Shape**: Rounded, friendly robot with soft edges (no sharp corners)
- **Size Proportions**: Slightly chibi/compact (big head, small body)
- **Color Scheme**:
  - Primary body: Gradient blue-purple (`#667EEA` to `#A855F7`)
  - Accents: Orange details (`#F97316`) for highlights
  - Screen/face: Soft glow effect
- **Materials**: Smooth plastic/metal look with subtle shine
- **Height**: Approximately 3:4 ratio (head to body)

#### Facial Features
- **Eyes**: Large, expressive circular eyes with sparkle highlights
  - Default: Friendly, slightly excited
  - Can change to express different emotions
- **Screen Face**: Optional glowing screen for displaying simple emotions (`:)`, `^_^`, etc.)
- **Antenna**: Single curved antenna on top with small glowing orb at tip (changes color based on state)

#### Body Details
- **Arms**: Short, rounded arms with 3-finger hands (simple, cartoon style)
- **Legs**: Stable base OR hovering with subtle propeller/rocket effect
- **Chest**: Circular glowing core (like a heart) in gradient colors
- **Details**: Small panel lines, rounded buttons, friendly tech aesthetic

### Robot States & Expressions

#### 1. **Welcome/Default** (Neutral friendly)
- Eyes: Wide open, happy
- Antenna orb: Blue glow
- Pose: Waving one hand
- Use: Landing page, first interactions

#### 2. **Encouraging** (Motivating)
- Eyes: Determined sparkle
- Antenna orb: Orange glow
- Pose: Thumbs up or fist pump
- Use: Starting tasks, skill gap analysis, project recommendations

#### 3. **Celebrating** (Excited)
- Eyes: Star-shaped excitement
- Antenna orb: Rainbow/multi-color glow
- Pose: Arms raised in victory, confetti around
- Use: Completed stories, portfolio improvements, achievements

#### 4. **Thinking** (Processing)
- Eyes: Slightly closed, contemplative
- Antenna orb: Purple pulsing
- Pose: Hand on chin or looking at floating code symbols
- Use: Loading states, analyzing repos

#### 5. **Guiding** (Helpful)
- Eyes: Focused, friendly
- Antenna orb: Green glow
- Pose: Pointing or holding a small sign/clipboard
- Use: Tutorials, tips, onboarding

#### 6. **Calm/Meditation** (Reassuring)
- Eyes: Gentle, relaxed
- Antenna orb: Soft blue-green glow
- Pose: Sitting cross-legged or hands together
- Use: Error states (but reassuring), taking breaks

## üéØ Usage Guidelines

### When to Use Each Mascot State:
- **Welcome**: Landing page hero, login page, account creation
- **Encouraging**: Skill gap analysis start, project recommendations, improvement suggestions
- **Celebrating**: Story completion, portfolio score improvement, achievements unlocked
- **Thinking**: Loading states, analyzing repositories, processing data
- **Guiding**: Onboarding steps, feature tutorials, help sections, tips
- **Calm**: Error states (friendly), break reminders, reassurance messages

### Illustration Placement:
- **Hero Section**: Abstract tech growth background + Welcome Craft Bot
- **Features**: Custom icon set in feature cards
- **Dashboard**: Developer workspace illustration or small Craft Bot states
- **Progress Tracking**: Growth plant progressions, playful charts
- **Empty States**: Craft Bot with encouraging scene
- **Onboarding**: Guiding Craft Bot with step indicators

### Animation Suggestions:
- Craft Bot: Gentle floating/bobbing animation (idle state)
- Antenna orb: Subtle glow pulse
- Plant illustrations: Gentle sway animation
- Confetti: Falling with physics on celebrations
- Progress bars: Fill with spring animation and growing leaves

## üìê Technical Specifications

### File Format Needs:
- **Craft Bot States**: SVG (scalable) + PNG exports at 1x, 2x, 3x
- **Wordmark Logo**: SVG primary, PNG fallbacks
- **Feature Icons**: SVG with 24x24, 32x32, 48x48 export sizes
- **Illustrations**: SVG where possible, high-res PNG for complex scenes

### Size Guidelines:
- **Craft Bot on hero**: ~300-400px height
- **Craft Bot inline (small)**: ~80-120px height
- **Feature icons**: 48px base size
- **Logo**: Minimum 120px width for wordmark

### Color Accessibility:
- All text on colored backgrounds should meet WCAG AA standards
- Use white text on vibrant colors
- Use dark blue (#0F172A) on light backgrounds
- Test illustrations on both light and dark modes
